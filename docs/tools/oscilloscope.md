## 1. 示波器选型指南

| 参数 | 入门/学生 | 专业嵌入式 | 高端 |
|------|----------|-----------|------|
| 带宽 | 50-100 MHz | 100-200 MHz | 350 MHz+ |
| 采样率 | 1 GSa/s | 1-2 GSa/s | 5 GSa/s+ |
| 通道数 | 2 | 4 | 4 |
| 推荐型号 | Rigol DS1054Z (~¥2,000) | Siglent SDS1204X-E (~¥3,500) | Keysight DSOX1204G (~¥8,000) |
| 存储深度 | 12 Mpts | 14 Mpts | 2 Mpts (但 Keysight 分段存储更强) |

**选型核心原则**：
- **带宽 ≥ 5 × 最高待测信号频率**（测 48MHz SPI CLK → 带宽 ≥ 250MHz）
- **至少 4 通道**（SPI 就要 CLK + MOSI + MISO + CS = 4 个信号）
- **强烈建议有串行总线解码功能**（I2C/SPI/UART/CAN 软件解码 → 直接看数据内容）

## 2. 探头使用技巧

### 2.1 ×1 与 ×10 探头

```
×1 探头:
  带宽: ~6-10 MHz (探头本身限制)
  输入电容: ~100 pF (会破坏高频信号的振荡条件!)
  适用: 低频信号、电源纹波测量

×10 探头:
  带宽: 100-500 MHz (示波器同带宽)
  输入电容: ~10-15 pF (远小于 ×1)
  适用: 所有数字信号、晶振测量、高速通信
```

::: danger 测晶振不能用 ×1 探头
用 ×1 探头测 OSC_IN 引脚时，探头自身的 100pF 电容会直接破坏晶振的起振条件。晶振可能停振或频率偏移。**必须用 ×10 探头，且探头尽量轻触引脚**。
:::

### 2.2 接地弹簧的使用

```
普通地线夹 (长 10cm 带鳄鱼夹):
  → 等效电感 ~100nH, 测高频信号时产生振铃和过冲
  → 适合: < 10MHz 信号

接地弹簧 (小弹簧套在探头尖端):
  → 等效电感 ~1nH, 接地面积极小
  → 适合: 所有高频/高速数字信号
  → 强烈推荐: 测量 SPI CLK, 晶振, 开关电源波形
```

## 3. 嵌入式常见测量场景

### 3.1 晶振起振验证

```
操作步骤:
  1. 探头 ×10, 接地弹簧
  2. 通道接 OSC_OUT (不要碰 OSC_IN，探头电容可能影响起振)
  3. 垂直灵敏度: 200mV/div
  4. 水平时基: 20ns/div (8MHz) 或 5ns/div (25MHz)
  5. 预期: 正弦波, 8MHz 峰峰值 ~0.8-1.5V (取决于晶振和负载电容)
  6. 如果波形很小或没有 → 晶振未正常振荡
```

### 3.2 电源纹波与噪声测量

```
操作步骤:
  1. 探头 ×1 (注意: 这是少数使用 ×1 的场景)
  2. AC 耦合 (隔离 DC 分量，只看纹波)
  3. 带宽限制: 20MHz (排除高频噪声)
  4. 垂直灵敏度: 10-50mV/div
  5. 水平时基: 10-100μs/div
  6. 预期: 纹波 < 50mVpp (好的电源) / < 100mVpp (可接受)
  7. 如果 > 200mVpp → 电源滤波不足或布局问题

关键: 探头要直接放在芯片 VDD 引脚旁边 (不是电源入口处)
```

### 3.3 I2C 信号质量

```
I2C 典型问题诊断:

上升沿过慢:
  ── 上拉电阻太大 (如 10kΩ 对于 400kHz Fast Mode)
  → 减小上拉到 2.2kΩ 或 1.5kΩ

下冲/过冲:
  ── 容性负载太大 (总线太长或器件太多)
  → 增加串联电阻 (22-50Ω) 在 SDA/SCL 靠近主机侧

交叉干扰 (Cross-talk):
  ── SCL 的边沿耦合到 SDA
  → PCB 上 SCL 和 SDA 走线间距 ≥ 3× 线宽，中间加地线
```

### 3.4 开关电源开关波形分析

```
关键测量点:
  ├─ SW (开关节点) 波形: 应该是干净的方波
  │   ├─ 振铃过大 → 缓冲电路 (Snubber) 不合适
  │   └─ 导通压降过高 → MOSFET Rds(on) 太大
  ├─ 输出纹波: 用 ×1 探头/AC 耦合测
  └─ 启动波形: 单次触发, 从 VIN 上电瞬间到 VOUT 稳定

触发设置: 上升沿触发, 单次模式 (Single), 垂直位置调到 VOUT
```

## 4. 波形分析与诊断

### 4.1 振铃 (Ringing) 分析

```
正常方波: ┌──┐    过冲振铃: ┌──╮╲
          │  │              │  ╰─
          └──┘              └────

振铃频率 ≈ 1 / (2π √(L_parasitic × C_load))

解决:
  1. 缩短走线长度 (减少 L_parasitic)
  2. 串联终端电阻 (22-50Ω) 在源端
  3. 如果走线无法缩短: RC 缓冲 (Snubber) 在负载端
```

### 4.2 地弹 (Ground Bounce) 检测

```
同时用两个通道:
  CH1: 芯片的 GND 引脚 (探头地线不接 — 用两个探头的差分)
  CH2: 数字输出信号

当多个输出同时从高到低切换时，CH1 上可以看到 mV 级乃至 V 级的毛刺
→ 说明 PCB 地去耦不够或布局有问题
```

## 5. 串行总线解码

现代数字示波器（即使是入门级 Rigol DS1054Z）都支持串行解码：

```
I2C 解码:
  1. 连接 CH1→SCL, CH2→SDA
  2. 触发类型: I2C → Start Condition
  3. 设置阈值电平 (通常 1.5V for 3.3V 系统)
  4. 解码结果显示: 地址、数据、ACK/NACK

SPI 解码:
  1. CH1→CLK, CH2→MOSI, CH3→MISO, CH4→CS
  2. 触发: CS 下降沿
  3. 设置 CLK 边沿 (上升沿/下降沿 → 取决于 SPI 模式)
  4. 解码显示十六进制数据

UART 解码:
  1. CH1→TX 或 RX
  2. 设置波特率 (手动输入)
  3. 设置极性: 空闲高 → 低电平为起始位
  4. 解码显示 ASCII 字符
```

## 6. 常见问题

| # | 问题 | 解决方法 |
|---|------|---------|
| 1 | 测出来的波形电压不对 | 检查探头是否在 ×10 档，示波器是否配置为对应倍数 |
| 2 | 高频信号看起来像正弦波 | 带宽不够: 方波包含大量谐波，带宽至少 5× 基频 |
| 3 | 波形上有 50Hz 的纹波 | 工频干扰: 用接地弹簧而非长地线夹；检查示波器接地 |
| 4 | 单次脉冲抓不住 | 设置合适的触发条件和触发电平；用 Single 模式而非 Auto |
| 5 | 探头一碰到晶振，晶振就停振 | 用了 ×1 探头 → 必须用 ×10；或者只碰 OSC_OUT，不碰 OSC_IN |
| 6 | 采样到的波形与实际不同 (混叠) | 采样率不够: 根据 Nyquist 定理，采样率 ≥ 2× 最高频率。实际需要 5-10× |
